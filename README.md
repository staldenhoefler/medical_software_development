# Medical Software Development

Coursework for the FHNW *Medical Software Development* module. Each numbered
folder is an exercise.

## Repository structure

| Folder | Topic |
|--------|-------|
| `01_development_gene_analyzer` | Gene analyzer development |
| `02_GC_Content_1` | GC content computation |
| `03_webapp` | Web application |
| `04_cost_estimation` | Cost estimation |
| `05_software_requirements_specification` | Software requirements specification (SRS) |
| `06_design_patterns` | Design patterns |
| `07_data_collection_service` | Data collection service (backend) |
| `08_sensor_overview` | Sensor overview |
| `09_data_collection_app` | Sensor data collection app + analysis notebook |
| `10_scientific_question` | Digital biomarker: tremor analysis |


Requires Python ≥ 3.12.
